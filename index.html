Virtual Lab: Exact & Non-Exact Differential Equations

An interactive learning environment for understanding differential equations, testing for exactness, and finding solutions

Aim

The aim of this virtual lab is to provide a comprehensive understanding of exact and non-exact differential equations - fundamental concepts in differential equations with applications in physics, engineering, and applied mathematics.

Through this lab, you will:

  • Understand the definition and properties of exact differential equations
  • Learn to test whether a given first-order differential equation is exact
  • Master the techniques for solving exact differential equations
  • Learn how to convert non-exact equations to exact ones using integrating factors
  • Apply these concepts to solve real-world problems in various scientific fields

Theory

A first-order differential equation of the form:

M(x, y)dx + N(x, y)dy = 0

is said to be exact if there exists a function F(x, y) such that:

∂F/∂x = M(x, y) and ∂F/∂y = N(x, y)

The necessary and sufficient condition for exactness is:

∂M/∂y = ∂N/∂x

When this condition holds, the general solution is given by F(x, y) = C, where C is an arbitrary constant.

If ∂M/∂y ≠ ∂N/∂x, the equation is non-exact. In such cases, we look for an integrating factor μ(x, y) such that when multiplied with the equation, it becomes exact. Common integrating factors depend only on x or only on y:

If (∂M/∂y - ∂N/∂x)/N is a function of x only: μ = e^∫f(x)dx
If (∂N/∂x - ∂M/∂y)/M is a function of y only: μ = e^∫g(y)dy

Pre-Test

Test your current knowledge about exact and non-exact differential equations before proceeding with the lab.

1. What is the condition for a differential equation Mdx + Ndy = 0 to be exact?

  • M = N
  • ∂M/∂x = ∂N/∂y
  • ∂M/∂y = ∂N/∂x
  • ∂M/∂x = ∂M/∂y

2. If an equation is exact, what form does its solution take?

  • y = f(x) + C
  • F(x, y) = C, where F is a potential function
  • x = g(y) + C
  • M(x, y) = N(x, y)

3. What is the purpose of an integrating factor?

  • To make a non-exact equation exact
  • To simplify the solution process
  • To find the derivative of the equation
  • To eliminate constants from the equation

Post-Test

After completing the lab, test your understanding with these questions.

1. Is (2xy + y²)dx + (x² + 2xy)dy = 0 exact?

  • Yes, because ∂M/∂y = 2x+2y and ∂N/∂x = 2x+2y
  • No, because ∂M/∂y = 2x and ∂N/∂x = 2x+2y
  • Yes, because M and N have the same variables
  • No, because M and N are different functions

2. What is the solution of the exact equation (2x + y)dx + (x + 2y)dy = 0?

  • x² + xy + y² = C
  • x² + 2xy + y² = C
  • 2x² + xy + 2y² = C
  • x² + xy + 2y² = C

3. For the non-exact equation y dx - x dy = 0, what is a suitable integrating factor?

  • μ = 1/x²
  • μ = 1/y²
  • μ = 1/(x²+y²)
  • μ = 1/(xy)
