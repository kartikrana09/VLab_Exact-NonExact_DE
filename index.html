Aim of the Virtual Lab

Learning Objectives

This virtual lab helps you understand exact and non-exact differential equations through interactive learning.

  • Understand the difference between exact and non-exact differential equations
  • Learn to identify exact differential equations
  • Master the method of solving exact differential equations
  • Learn to convert non-exact equations to exact using integrating factors
  • Apply these methods to solve real-world problems

Theoretical Background

Exact Differential Equations

A differential equation of the form M(x,y)dx + N(x,y)dy = 0 is exact if:

M/y = N/x
Example: (2xy + y)dx + (x + 3xy)dy = 0

The general solution is F(x,y) = C, where C is a constant.

Rules for Finding Integrating Factors

  1. If (M/y - N/x)/N is function of x only:
    IF = ef(x)dx
  2. If (N/x - M/y)/M is function of y only:
    IF = eg(y)dy
  3. If equation is homogeneous and xM + yN 0:
    IF = 1/(xM + yN)
  4. If the D.E is in the form yf(x,y)dx+xf(x,y)dy=0
    IF = 1/(xM - yN)
